== Description ==

Menu Import Export by [ReCorp][recorp] is the perfect nav menu plugin for any websites. Menu Import Export plugin will help you a lot. Works simply within one click!
It supports also uber menu plugin. You can import menus to same or cloned site. You can also import the menus to other site if menu items were created with `custom links`.

== Installation ==

= Minimum Requirements =

* WordPress 3.6 or greater
* PHP version 5.3 or greater

= Installation =

* Install `Menu Import & Export` either via the WordPress.org plugin repository or by uploading the files to your server.
* Activate `Menu Import & Export` plugin.
* Go to wp-admin > Appearance > Menus and you will see "Export & Import" button.


== Frequently Asked Questions ==
= Can I import menus on other site ? =
You can import menus but need to be exactly same page or post id in both site.
